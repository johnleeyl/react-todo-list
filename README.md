# React To-Do List

A simple To-Do List application built with **React** and **Vite** as my first React project. This project was created to practice React fundamentals such as state management, event handling, rendering lists, and updating the UI based on user interactions.

## Features

- ✅ Add new tasks
- ✅ Delete existing tasks
- ✅ Move tasks up the list
- ✅ Move tasks down the list
- ✅ Prevent adding empty tasks
- ✅ Responsive and clean user interface

## Built With

- React
- JavaScript (ES6+)
- Vite
- CSS3

## Concepts Practiced

Throughout this project, I learned and applied:

- React Functional Components
- `useState` Hook
- Event Handling (`onClick`, `onChange`)
- Controlled Components
- Rendering Lists using `map()`
- Updating State Immutably
- Array Manipulation
  - `filter()`
  - Spread Operator (`...`)
  - Swapping array elements
- Conditional Logic

## Getting Started

Clone the repository

```bash
git clone https://github.com/johnleeyl/react-todo-list.git
```

Navigate into the project

```bash
cd react-todo-list
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
